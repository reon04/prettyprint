# prettyprint

pretty print is a python module that prettifies the printing of the print function (everthing is printed in lower case)

## installation
```
git clone https://github.com/reon04/prettyprint.git
cd prettyprint
pip install --user .
```
notes:
- choose pip or pip3 accordingly
- when using a venv the option `--user` is not needed

### uninstallation
```
pip uninstall prettyprint
```

## usage
simply import the module as any other python module, the print function can then be used like normal (but prints everything in lowercase yay)
```
import prettyprint
```
